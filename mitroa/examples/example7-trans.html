Υπολογισμός παραμορφώσεων#

Είναι \(K_{UU}\cdot U_U = P_U + R_U\), με \(R_U=0\)

οπότε \(K_{UU}\cdot U_U = P_U\)

και τελικά \(U_U = K_{UU}^{-1}\cdot P_U\)

U_U = K_UU.inv() * P_U
round_expr(U_U, 7)
\[\begin{split}\displaystyle \left[\begin{matrix}0.001071\\-3.58 \cdot 10^{-5}\\-0.0025288\\0.0009766\\3.58 \cdot 10^{-5}\\0.0035075\end{matrix}\right]\end{split}\]

Υπολογισμός αντιδράσεων στήριξης#

Είναι \(K_{RU}\cdot U_U = P_R + R_R\)

οπότε \(R_R = K_{RU}\cdot U_U - P_R\)

R_R = K_RU*U_U - P_R
round_expr(R_R, 2)
\[\begin{split}\displaystyle \left[\begin{matrix}77.43\\57.28\\-62.2\\-177.43\\-57.28\\191.32\end{matrix}\right]\end{split}\]

Επίλυση/έλεγχος με το OpenSees#

ops.wipe()
ops.model('basic', '-ndm', 2, '-ndf', 3)

ops.node(1, 0., 0)
ops.node(2, 0, Ly)
ops.node(3, Lx, Ly)
ops.node(4, Lx, 0)

ops.fix(1, 1, 1, 1)
ops.fix(4, 1, 1, 1)

# opsv.plot_model()
# plt.title('plot_model before defining elements')

ops.geomTransf('Linear', 1)

# elements
ops.element('elasticBeamColumn', 1, 1, 2, Ac, E, Ic, 1)
ops.element('elasticBeamColumn', 2, 2, 3, Aw, E, Iw, 1)
ops.element('elasticBeamColumn', 3, 3, 4, Ac, E, Ic, 1)


# loads
ops.timeSeries('Constant', 1)
ops.pattern('Plain', 1, 1)

# ops.eleLoad('-ele', 1, '-type', '-beamPoint', -F_load, 0.5)
ops.load(2, F2, 0, M2)
ops.load(3, F3, 0, M3)



# analysis
ops.constraints('Transformation')
ops.numberer('RCM')
ops.system('BandGeneral')
ops.test('NormDispIncr', 1.0e-6, 6, 2)

ops.integrator('LoadControl', 1)
ops.analysis('Static')
ops.analyze(1)
